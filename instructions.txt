Set-up
--> Install pytest:
    - In your command line:
        >> pip install -U pytest
        >> pytest --version
    - Your should have pytest version 3.x.y
--> Install future (for testing):
    - In your command line:
        >> pip install future
    - This will be used for testing where:
        from __future__ import absolute_import
